


choco upgrade autohotkey.install
choco upgrade microsoft-windows-terminall

run ahk, press win+` (win+tilde) to open windows terminal

consider adding
    "initialPosition": "0,0",
    "initialCols": 234,
into the settings json to have the window start at the top of the main monitor and span across a 1920x1080 screen
https://docs.microsoft.com/en-us/windows/terminal/customize-settings/startup
for more info
